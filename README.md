- 추가적으로 프로젝트 진행 상황 고려하여 도메인설정도 하려함.
- 아직 디자이너분께 figma를 받지 못함 => `Notion Api 명세서`에 따라서 임의로 서버로 비동기적인 데이터 전달 및 받아오는 전역 상태를 구성중.
